# 首页 UI 优化分析

> 分析视角：产品经理 + 学生用户
> 分析范围：`src/pages/home/index.tsx` 及相关组件
> 日期：2026-03-18

---

## 一、产品经理视角

### 1. 信息架构问题

**数据统计卡布局反直觉**
`overviewStats` 卡片的渲染顺序是 label 在上、数值在下（见 index.tsx:1204-1228）。用户视线先看到"上岸率"这个标签，再看到"92.3%"这个数字，与"先看到大数字、再理解含义"的阅读习惯相反。建议数值在上、label 在下。

**Hero 双模式导致体验分裂**
代码中存在 `heroHasBackgroundImage` 分支（index.tsx:1038），有背景图和无背景图时渲染完全不同的样式。这意味着 CMS 配置一张图就会切换整个 Hero 的视觉风格，产品行为不可预期，维护成本高。

**"学习支持"板块信息密度过低**
`advantages` 卡片只渲染了 `item.title`，`item.desc` 被完全忽略（index.tsx:1435-1445）。两个卡片各占 48.6% 宽度，却只显示一行标题，空间浪费严重，没有传递实质价值。

**快捷入口宽度过窄**
4 个快捷入口各占 23% 宽度（index.tsx:1241），在 320px 宽的小屏设备上，每个入口只有约 73px，图标 + 文字容易拥挤或截断。

### 2. 转化漏斗问题

**首屏主 CTA 动作选择错误**
Hero 区主按钮文案是"查看上岸学员案例"（contentFallback.ts:22），这是一个**信任建立**动作，不是**转化**动作。对于首次访问的学生，最强的转化动作应该是"免费咨询"或"了解课程"。上岸案例应作为辅助信任背书，而非主 CTA。

**底部 CTA 卡片说服力不足**
`cta.desc` 在 fallback 数据中为空字符串（contentFallback.ts:59），`PageCtaCard` 在 desc 为空时直接跳过渲染（PageCtaCard/index.tsx:43-58），导致底部转化卡片只有标题和按钮，缺乏任何说服性文案。

**全页转化入口单一**
整个首页只有 Hero 区 1 个主 CTA 和底部 1 个 CTA 卡片，且两者文案相似（"查看上岸学员案例" / "预约咨询"）。缺少在内容浏览过程中的中间层转化触点。

### 3. 视觉一致性问题

**双色系并存**
`ui.ts` 定义的全局 accent 色是紫色 `#5b4dff`（ui.ts:11），但首页 `homeTone` 使用蓝色系 `#0284c7`（index.tsx:1020），Hero 渐变使用 `#4facfe → #00f2fe`（index.tsx:1038）。三套蓝/紫色系在同一页面共存，没有统一的品牌色主张。

**方向卡片颜色不受控**
`tagColor`、`headerBackground`、`iconColor` 全部来自 CMS 数据（index.tsx:127-131），如果运营配置了不协调的颜色，首页会出现色彩冲突，且前端无兜底约束。

## 二、学生用户视角

### 1. 第一眼不知道这是干什么的

- Hero chip 文案"江苏省专转本专业辅导"是机构的自我定位描述，不是学生的问题陈述
- 学生进来的第一个问题是"我能不能考上"，而不是"这个机构有多权威"
- 建议 Hero 文案从学生视角出发，例如"护理/助产背景，想转本？" 这类问题式开头更能引发共鸣

### 2. 关键决策信息缺失

学生在首页做决策需要的信息，目前都找不到：
- **价格**：没有任何收费区间或"了解价格"的入口
- **时间**：没有开课时间、报名截止日期等时效性信息
- **地点**：地址信息只在 fallback 数据里有（"江苏省淮安市"），首页没有展示
- **课程时长**：方向卡片没有展示课程周期

### 3. 新用户操作路径不清晰

快捷入口"每日一题/模拟题/错题本"是**留存功能**，面向已报名的在读学生。对于刚进来的新用户，这些入口没有意义，反而会造成困惑（"我还没报名，为什么要做题？"）。

首页缺少一条清晰的新用户引导路径：**了解方向 → 查看案例 → 咨询报名**。

### 4. 信任建立手段单薄

- 学习环境图片只有 2 张，高度约 94px（188rpx），太小，无法建立真实感
- 图片使用 `imageSeed` 占位（contentFallback.ts:53-54），实际展示的可能是随机占位图而非真实校区照片
- 上岸率"92.3%"是核心数据，但只在统计卡里用小字展示，没有在 Hero 区做视觉强调

## 三、优先级建议

| 优先级 | 问题 | 影响 |
|--------|------|------|
| P0 | Hero 主 CTA 文案改为转化导向 | 直接影响咨询转化率 |
| P0 | 底部 CTA 补充说服性文案 | 直接影响咨询转化率 |
| P1 | 数据统计卡数值/label 顺序调整 | 影响关键数据的传达效率 |
| P1 | "学习支持"卡片补充 desc 展示 | 影响机构价值传达 |
| P1 | 统一首页色系，消除双色系并存 | 影响品牌一致性 |
| P2 | 新用户引导路径设计 | 影响新用户转化漏斗 |
| P2 | 快捷入口区分新老用户场景 | 影响新用户体验 |
| P2 | 学习环境图片尺寸和真实性 | 影响信任建立 |
| P3 | Hero 双模式重构为单一模式 | 影响维护成本 |
| P3 | 方向卡片颜色前端兜底约束 | 影响视觉稳定性 |
